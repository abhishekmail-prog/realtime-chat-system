# MERN Chat Application

A modern real-time chat application built with the MERN stack.
Vibe Chat provides secure authentication, live messageing using
Socket.IO, profile management with Cloudinary image uploads,
and a clean responsive user interface.


- Status:  🚧 In Actively under development.

---

## Features

### Authentication

- User Signup & login
- JWT Authentication
- Protected Routes
- Authentication Check

### Real-Time Chat

- One-to-One Messaging
- Live Communication using Socket.IO
- Online User Status
- Instant Message Updates

### User Profile

- Edit Name & Bio
- Upload Profile Picture
- Cloudinary Image Shortage
- Persistent User Profiles

### User Interface

- Responsive Design
- Modern Dark Theme
- Purple UI
- Mobile-friendly-layout
