# MERN Chat Application

A real-time chat application built using the MERN stack (MongoDB, Express, React, Node.js) with WebSocket-based communication.

## Features
- User Authentication (JWT)
- Real-time messaging (Socket.IO)
- One-to-one chat
- Scalable backend architecture

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- Real-time: Socket.IO
